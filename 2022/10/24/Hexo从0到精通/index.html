前言：
随着互联网浪潮的翻腾，国内外涌现出越来越多优秀的社交网站让用户分享信息更加便捷。然后，如果你是一个不甘寂寞的程序猿（媛），是否也想要搭建一个属于自己的个人网站，如果你曾经或者现在正有这样的想法，请跟随这篇文章发挥你的Geek精神，让你快速拥有自己的博客网站，写文章记录生活，享受这种从0到1的过程。
下面正式从零开始搭建年轻人的第一个网站。
一、安装node.js和git（百度自寻）
首先下载node.js并安装，安装的时候一直点Nest就行

1、其次配置node.js的环境变量的具体步骤（以win11为例）如下：

  • 设置- ->系统- ->系统信息- ->高级系统设置- ->高级- ->环境变量

  • 复制nodejs目录路径(F:\Program Files\node-v8.11.1-win-x64)然后选中用户变量中的Path,单击编辑- ->新建- ->输入刚复制的地址 - ->确认完成

  • 重启cmd并输入（如若不重启将无法查到版本号）node -v,出现下图则说明配置成功:


2、下载git，一路Nest就行，不在这废话了

  • 安装完之后cmd输入git –version检查有没有安装成功，出现下图则说明配置成功:


二、安装Hexo
1、cmd命令行输入npm install hexo-cli -g回车进行安装

  • 安装完成后，输入hexo -v检查是否安装成功，出现下图则说明安装成功:


三、搭建GitHub仓库
1、GitHub账号注册

  • 进行登录,用户名尽量起个简短一点的

2、创建仓库

  • 回到GitHub.com主页面，点击new新建仓库，仓库名必须是你的用户名加上.GitHub.io（例如我的用户名是yiyan20202020，就需要填yiyan20202020.GitHub.io）下面Add a README file记得打勾哦，最后点击下面绿色 Create repository 就完成啦


四、生成SSH keys
1、检查ssh

  • 进入任何一个文件夹右键，选择Git Bath Here,输入ssh，显示以下内容，则为已经安装过了进入下一步


2、关联SSH keys

  • 输入ssh-keygen -t rsa -C "你的邮箱"中间有空格别忘了，然后回车四次

  • win+e打开文件管理器，找到c盘里的User，选择你的管理员/用户文件夹，找到.ssh文件

  • 打开.ssh文件，并打开id_rsa.pub文件复制公钥（可以用文本打开）ctrl+a，ctrl+c

  • 返回GitHub，点击右上角，打开设置，找到SSH and GPG keys，添加SSH key，title随便填，把刚才复制的公钥key粘贴到里面，然后点击Add SSH keys

  • 返回刚刚的命令行输入ssh -T git@GitHub.com并输入yes然后回车，出现以下内容则说明绑定成功


五、本地生成博客内容
1、创建文件夹

  • 在任何一个盘里创建一个文件夹，命名随便
  • 进入刚刚创建的文件夹右键点击Git Bash Here

2、初始化hexo

  • 输入hexo init，回车初始化hexo，如果出现err错误就重新输入然后回车，直至安装成功，显示以下内容

  • 然后输入hexo s，本地生成页面，复制并在浏览器打开，hexo在本地搭建就算完成了


六、把博客发布到互联网
1、修改配置

  • 打开你的仓库项目点击code，复制你的项目链接

  • 打开之前创建的文件夹，并打开_config.yml文件，修改最下面的配置，中间的链接粘贴你自己的仓库项目链接

  • 改完之后ctrl+s保存

2、安装自动部署发布工具

  • 在博客文件夹右键点击Git Bash Here,输入命令行npm install hexo-deployer-git --save回车安装


3、生成页面并上传

  • 输入hexo g生成页面

  • 输入hexo d上传到GitHub仓库，期间可能会出现以下报错

  • 遇到报错不要急，看到最前面两行没，把邮箱改成注册GitHub的邮箱，把name改成你的GitHub用户名，依次运行它，然后GitHub网页会跳转一个授权，授权一下就好了，重新输入hexo d显示以下内容为上传成功

  • 上传完成之后就可以用github.io访问你的博客了
